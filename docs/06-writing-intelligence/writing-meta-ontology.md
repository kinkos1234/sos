# 글쓰기 메타온톨로지 × 메타엣지 × 분석공간

> 출처: 3월 6-7일.docx — "글을 잘쓰는 방법"을 분석공간을 활용해 정의

---

## 메타온톨로지 3가지 공리

**A1. 意味의 단일성 (Semantic Monism)**
- 모든 문장은 하나의 핵심 의도를 전달해야 한다
- 검증: "이 문장이 없으면 글의 의미가 바뀌는가?" → YES면 필수

**A2. 구조의 투명성 (Structural Transparency)**
- 읽는 사람이 글의 골격을 3초 안에 파악
- 제목/첫문단/구간 제목만으로 전체 구조가 보여야 함

**A3. 시간의 선물 (Temporal Gift)**
- "읽는데 10분 걸렸는데 가치가 15분어치인가?"

## 7+1개 분석공간

### SPACE 1: SEMANTIC (의미)
- concept_clarity, word_precision, metaphor_effectiveness, ambiguity_level, semantic_depth, lexical_variety, jargon_density
- 핵심 규칙: `concept_clarity(≥80) AND ambiguity_level(≤20) → 의미 전달 성공`

### SPACE 2: STRUCTURAL (구조/위계)
- hierarchy_clarity, flow_smoothness, segment_coherence, information_chunking, section_balance, logical_progression, visual_scannability
- `information_chunking` 최적점 존재 (너무 나누면 flow_smoothness 해침)

### SPACE 3: TEMPORAL (시간/리듬)
- sentence_rhythm, pacing_speed, suspense_curve, revelation_timing, climax_positioning, reading_momentum, time_investment_value
- `actual_perceived_speed = pacing_speed × (0.3 + 0.7 × sentence_rhythm/100)`

### SPACE 4: CLUSTER (패턴/군집)
- theme_consistency, example_diversity, pattern_recognition, categorization_clarity, repetition_effectiveness, grouping_density
- 반복 최적점: 3-4회 (≤3=강조, >5=지겨움)

### SPACE 5: PROBABILISTIC (불확실성/기대)
- expectation_setting, uncertainty_tolerance, surprise_factor, confidence_level, assumption_transparency, reader_prediction_accuracy
- 역설: uncertainty_tolerance(높음) + confidence_level(높음) = 흥미 유발

### SPACE 6: CAUSAL (인과/논리)
- argument_strength, evidence_quality, premise_validity, logical_fallacy_free, causal_chain_clarity, counterargument_handling
- logical_fallacy_free(너무 높음) → 오히려 의심 (humility signal 필요)

### SPACE 7: CROSS_SPACE (통합/일관성)
- all_spaces_alignment, voice_consistency, tone_stability, message_reinforcement, narrative_integrity, reader_experience_design, authenticity_score
- **가장 중요**: 각 SPACE별 점수가 높아도 all_spaces_alignment가 낮으면 "뭔가 어색한 글"

### SPACE 감정: EMOTIONAL (감정/영향)
- emotional_resonance, empathy_trigger, inspiration_factor, tension_relief, catharsis_opportunity, joy_quotient

## 정량적 정의

```
TOTAL_SCORE =
    semantic(0.15) + structural(0.15) + temporal(0.12) + cluster(0.10)
  + probabilistic(0.08) + causal(0.15) + coherence(0.15) + emotional(0.10)

 0-40  : "배워야 함" (초급)
40-60  : "괜찮음" (중급)
60-75  : "좋은 글" (상급)
75-85  : "우수한 글" (전문가)
85-100 : "뛰어난 글" (마스터)
```

### 도메인별 가중치 조정
- **과학 논문**: causal × 0.25, semantic × 0.20
- **에세이/수필**: emotional × 0.20, voice × 0.18
- **뉴스 기사**: structural × 0.22, semantic × 0.18

## 사람들이 좋아하는 글 — 메타엣지 법칙

1. 명확함(SEMANTIC) → 신뢰도 (+0.81), BUT 95+ → 오히려 의심
2. 리듬감(TEMPORAL) → 공유 욕구 (+0.79)
3. 진정성(authenticity) → 장기 영향력 (+0.87)
4. 논리성 + 감정성 → 행동 유도 (+0.91) — "머리로 납득, 가슴으로 동의"
5. 기대 설정(PROBABILISTIC) → 완독률 (+0.83)
6. 예시 다양성(CLUSTER, 3개 이상) → 이해도·기억력 향상
7. 반박 가능성 인정 → 신뢰도 (+0.84) — 완벽하지 않은 주장이 오히려 신뢰

## 실무 체크리스트 (글 완성 전)

- [ ] SEMANTIC: 핵심 용어 3개를 1문장으로 정의했는가?
- [ ] STRUCTURAL: 목차/제목만 읽어도 구조가 보이는가?
- [ ] TEMPORAL: 문장 길이 변동성 > 30인가?
- [ ] CLUSTER: 같은 주제의 예시가 3개 이상인가?
- [ ] PROBABILISTIC: "이 글이 뭐 하려는 글?" 명확한가?
- [ ] CAUSAL: 각 주장에 근거가 있는가?
- [ ] COHERENCE: 글 전체가 하나의 "톤"으로 일관되는가?
- [ ] EMOTIONAL: 읽는 사람이 공감할 포인트가 있는가?
